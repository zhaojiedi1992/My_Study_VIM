编辑文本
================================================================================

----------------------------------------------------------
插入
----------------------------------------------------------
- `i`: 从光标前开始插入字符
- `I`:  从行首开始插入字符
- `a`:  从光标后开始插入字符
- `A`:  从行尾开始插入字符
- `o`:  在当前行之下另起一行, 开始插入字符
- `O`:  在当前行之上另起一行, 开始插入字符


----------------------------------------------------------
编辑文本
----------------------------------------------------------
- `r`:  替换当前字符
- `R`:  在 ESC 按下之前，替换多个字符
- `J`:  将下一行合并到当前行, 并在两部分文本之间插入一个空格
- `c$`:  将光标处到行尾删除, 然后进入插入模式
- `u`:  撤销
- `Ctrl + r `:  重做（取消撤销）
